# Churn-Prediction
# Project Title: Bank Customer Churn Prediction using Machine Learning:
# overview:
This project aims to predict whether a bank customer is likely to churn using machine learning. It analyzes customer details such as demographics, account information, and banking behavior to identify customers at risk. Based on the prediction, banks can take proactive retention measures like personalized offers and better customer support. This helps reduce customer churn, improve customer satisfaction, and increase the bank's overall profitability.
